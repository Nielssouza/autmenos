Teste

Esta pagina existe para ajudar a equipe a validar rapido se os templates do Django estao funcionando e para deixar os primeiros links do projeto sempre visiveis.

Abrir admin JWT endpoint Login DRF

Primeiros passos

  • Ative o ambiente virtual.
  • Instale as dependencias do projeto.
  • Crie o arquivo .env usando o exemplo.
  • Rode as migrations antes de subir o servidor.

Comandos uteis

Use estes comandos no terminal para validar o ambiente:

python manage.py migrate python manage.py createsuperuser python manage.py runserver

Rotas disponiveis

  • / pagina inicial com orientacoes
  • /admin/ painel administrativo
  • /api/token/ gerar token JWT
  • /api/token/refresh/ renovar token
  • /api-auth/ login do browsable API

Quando o projeto crescer, esta pagina pode virar um portal interno ou ser substituida pela interface principal do sistema.
